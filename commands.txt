<id>                              | Run <id> on current tests
main|bf|ch|gen <id>               | Open .cpp of main|bf|ch|gen of <id>
debug main|bf|ch|gen <id>         | Open .cbp of main|bf|ch|gen of <id>
add <id>                          | Add test to <id>
keep <id> <num>                   | Keep first <num> tests of <id> and delete the rest
rm <id> <num>                     | Delete last <num> tests of <id>
tl <id> <tl>                      | Set TL of <id> to <tl>
stress <id> <num>                 | Stress <id> on <num> tests
check <id> <num>                  | Check <id> on <num> tests
path                              | Copy path of last used <id> to clipboard
cls                               | Clear screen
help                              | Print this guide
exit                              | Exit contest
